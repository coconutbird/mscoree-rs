Add Additional Hosting interfaces for CLR control

This adds hosting control interfaces:
- ICLRControl, IHostControl - Host/CLR control exchange
- ICLRGCManager, ICLRGCManager2 - GC configuration
- ICLRPolicyManager - Runtime policy control
- ICLRErrorReportingManager - Watson error reporting
- ICLRHostProtectionManager - Code access security
- ICLRDebugging - Debug process attachment
- ICLRDebuggingLibraryProvider (1-3) - Debugging library provision
- ICLRStrongName - Strong name key management and signing
- ICLRTask, ICLRTask2 - CLR task representation
- IHostTask - Host task implementation
- ICLRTaskManager, IHostTaskManager - Task/thread management
- CLR_DEBUGGING_VERSION, CLR_DEBUGGING_PROCESS_FLAGS structs/enums
