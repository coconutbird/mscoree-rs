Add Fusion/Assembly binding interfaces (legacy .NET Framework)

This adds the Global Assembly Cache (GAC) interfaces:
- IAssemblyCache - Query, install, and uninstall assemblies from GAC
- IAssemblyCacheItem - Assembly cache installation item
- IAssemblyName - Assembly identity representation
- IAssemblyEnum - Enumerate assemblies in the cache
- IInstallReferenceItem, IInstallReferenceEnum - Install references
- ASM_CACHE_FLAGS - Cache types (ZAP, GAC, DOWNLOAD, ROOT)
- ASM_DISPLAY_FLAGS - Assembly display name formatting
- ASM_NAME - Assembly name property enumeration
- ASSEMBLY_INFO - Assembly information structure
