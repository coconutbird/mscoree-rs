Rename interface files to follow clr_ prefix convention

Rename files containing ICLRxxx interfaces to use the clr_ prefix:
- metahost.rs -> clr_metahost.rs
- runtime_host.rs -> clr_runtime_host.rs
- runtime_info.rs -> clr_runtime_info.rs
- data_target.rs -> clr_data_target.rs

This ensures consistent naming across all interface files:
- clr_* for ICLRxxx interfaces
- cor_debug_* for ICorDebugxxx interfaces
- cor_profiler_* for ICorProfilerxxx interfaces
- xclr_data_* for IXCLRDataxxx interfaces
- metadata_* for IMetaDataxxx interfaces
