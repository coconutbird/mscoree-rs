Add ICorProfiler family interfaces for .NET profiling

This adds the core profiler interfaces from mscoree:
- ICorProfilerCallback - ~50 callback methods for profiler events
  (AppDomain, Assembly, Module, Class, Thread, JIT, GC, Exception events)
- ICorProfilerInfo - Query interface for profilers (~30 methods)
- ICorProfilerInfo2 - Extended query interface with stack snapshots,
  generics support, and additional inspection methods
- CorPrfMonitor enum - Profiler event flags
- CorPrfGcGeneration enum - GC generation values
